# Instagram Playbook

## Role of Instagram
Instagram is a conversion-support channel. It is not only for visibility. Content should serve specific funnel jobs that move qualified audiences toward the free masterclass and eventually the paid course.

## Core Carousel Archetypes

### 1. Awareness
Goal: create recognition and stop the scroll.
Use for: paradigm-shift content, hidden tensions, emotionally resonant problem recognition.

### 2. Education
Goal: teach a useful distinction and build trust.
Use for: mistakes, reframes, conceptual clarity, structured takeaways.

### 3. Objection
Goal: reduce resistance.
Use for: pre-sale or pre-registration friction, common beliefs that block action.

### 4. Conversion
Goal: generate direct registration.
Use for: masterclass invitation, learning outcomes, urgency, offer bridge.

## Mandatory Brief Before Production
Every asset should define:
- title
- funnel stage
- objective
- audience
- core tension
- key takeaway
- CTA
- linked offer
- visual tone
- slide count

## Default Slide Formula
6-slide default:
1. Hook
2. Problem or tension
3. Insight
4. Reframe
5. Takeaway
6. CTA

Alternative education formula:
1. Hook
2. Context
3. Point 1
4. Point 2
5. Point 3
6. Synthesis
7. CTA

## CTA Logic
- Awareness: soft CTA
- Education: bridge CTA
- Objection: curiosity-to-action CTA
- Conversion: direct CTA, usually registration-focused

## Design Rules
- one idea per carousel
- one main message per slide
- clear mobile readability
- consistent CTA slide treatment
- clinical and intelligent tone
- minimal clutter

## Agent Workflow
1. Classify the idea by funnel stage.
2. Build the strategy brief.
3. Write slide-by-slide copy.
4. Create design brief from template rules.
5. Run QA before publishing.

## QA Questions
- Does the first slide stop the scroll?
- Is the funnel role clear?
- Is the CTA appropriate?
- Is the language on-brand?
- Is the design readable?
- Does it feel like MotusDAO rather than generic therapy content?
