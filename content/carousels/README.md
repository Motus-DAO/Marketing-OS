# Carousel Packs

Each folder contains a 7-slide review pack using this structure:
1. Cover image + text
2. Gradient slide + text
3. Cover image + text
4. Gradient slide + text
5. Cover image + text
6. Gradient slide + text
7. Cover image + text / CTA

Typography:
- Jura Bold for large text
- Inter for supporting text
- gradient highlight on key words
