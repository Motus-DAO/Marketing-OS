# Production Packet — Instagram Carousel

## Asset
5 errores comunes al pasar de consulta presencial a online

## Objective
Create first-draft visual assets as HTML/CSS or SVG-based slide mockups for iteration.

## Funnel stage
Education

## Platform
Instagram Carousel

## Canvas
1080 x 1350 px per slide

## Brand direction
Use MotusDAO visual language v1:
- cinematic editorial tech-clinical
- premium, serious, clinically intelligent
- dark graphite / deep blue-black backgrounds
- strong editorial hierarchy
- one dominant idea per slide
- no wellness aesthetic
- no meme styling
- no clutter

## Style references
Use the visual logic of the user’s option 1 and option 4 as the primary hybrid direction.
- Option 1: emotional/editorial portada, strong cinematic contrast
- Option 4: cleaner modular internal slides, more structured clarity

## Typography
- Headline font: Jura Bold
- Secondary/body font: Inter

## Text styling rules
- Big text uses Jura Bold
- Small text and supporting copy use Inter
- Use gradient highlighting only on key phrase(s)
- Do not apply gradients to full paragraphs

## Gradient rule
Allowed for emphasis on main thought phrases only.
Suggested gradient direction:
- cyan -> electric blue -> soft violet
Alternative:
- icy blue -> teal -> indigo

## Composition rules
### Cover slide
- strongest visual energy
- 1 dominant image/texture/background
- large title with clear contrast
- minimal secondary copy

### Internal slides
- strong numbering system for each error
- short headline
- short explanatory copy
- asymmetrical but controlled composition
- support visual should never overpower readability

### CTA slide
- cleanest slide
- strongest spacing
- premium and restrained
- clear masterclass CTA
